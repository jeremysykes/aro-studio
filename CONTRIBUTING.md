CONTRIBUTING

Thanks for your interest in Aro Studio.

This project is currently proprietary and not accepting public code contributions.

You MAY:
- Open issues for bugs, questions, and feature requests.
- Propose ideas and workflow improvements.

You MAY NOT:
- Submit pull requests containing code changes unless explicitly invited.
- Publish modified versions of this repository or any derived works.
- Use any part of this code in production or commercial projects without permission.

If you want to collaborate or license the software:
- Open an issue titled "Licensing / Collaboration" with a brief description of your use case.
- Include whether you need: evaluation extension, commercial license, or partnership.

Security:
If you discover a security issue, do not disclose it publicly. Open a private channel by creating
an issue titled "Security" and request a secure contact method.
